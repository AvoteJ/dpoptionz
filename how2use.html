DPOPz (Double Play Optionz)란?

DPOPz는, Double Player들을 위한 배치표 자료 통계 사이트로, 의견 제안을 통해 다른 DPer들, 특히 초심자들을 위한 배치 가이드라인으로써 사용하기 위해 제작된 페이지입니다.

웹 페이지 사용 방법

  1. Top
    해당 페이지의 검색 기능을 통해 곡명을 검색하면 즉시 해당 곡에 사용된 배치들의 분포도가 표시됩니다. Vote 버튼을 통해 자신의 의견도 즉시 제출이 가능합니다. (Vote를 위해서는 가입이 필요합니다.)
  2. Table
    데이터가 수집된 모든 데이터의 열람이 가능합니다. 현재 집계중인 데이터는 아래와 같습니다.
    레벨집계 대상집계 여부
    ☆12AC 전체 보면집계
    ☆11AC 전체 보면집계
    ☆10AC의 ANOTHER / LEGGENDARIA 보면보류*
    ☆9AC의 ANOTHER / LEGGENDARIA 보면보류*
    *추후 추가 예정
  3. Report
    해당 사이트에 오류가 있을 경우, 해당 페이지로 보고를 부탁드립니다.

Vote 기능에 대하여 (회원 전용)

Log In 기능이나 Vote 버튼을 누르면 로그인이 활성화되며, 가입 절차를 통해 투표 기능을 사용할 수 있습니다. Vote를 통해 배치가 제출되면, 약 3~7일 내에 DB에 반영되어, 통계에 적용됩니다. ※해당 페이지는 전문적인 보안처리가 되지 않은 Vercel 시스템을 활용한 페이지입니다. 개인이 자주 사용하는 ID 또는 PASSWORD 사용을 지양해주시기 바랍니다. ※회원 가입 시 작성되는 IIDX ID 및 e-mail은 본인 확인을 위한 수단으로만 사용됩니다. ※IIDX ID당 1개의 ID만 가입이 가능하며, 중복 가입은 허용되지 않습니다. 가입 양식을 작성 후 3~7일 이내에 관리자가 확인 후 계정 승인이 되어야 투표 기능 사용이 가능합니다.

DPOPz (Double Play Optionz)とは？

DPOPzは、Double Playerのための配置統計サイトです。 利用者の意見次第で、他のDPerたち、特に初心者のための配置ガイドラインとして 活用することを目的として制作されたページです。

ウェブページの使い方

  1. Top
    検索機能で曲名を検索すると、その曲で使用された配置の分布が表示されます。 Voteボタンから自分の意見もすぐに投稿できます。 (Voteには登録が必要です。)
  2. Table
    収集済みの全データを閲覧できます。現在集計中のデータ対象は以下の通りです。
    レベル集計対象集計状況
    ☆12AC 全譜面集計中
    ☆11AC 全譜面集計中
    ☆10ACの ANOTHER / LEGGENDARIA 譜面保留*
    ☆9ACの ANOTHER / LEGGENDARIA 譜面保留*
    *今後追加予定
  3. Report
    サイトに不具合がある場合は、こちらのページからご報告をお願いします。

Vote機能について (サインアップ必)

Log Inボタンや、Voteボタンをクリックするとログイン機能が活性化され、Vote機能を使うにはサインアップの手続きが必要となります。 Voteでオプションの意見を提出した後、約３～７日のうちDBに反映され、統計に適用されます。 ※当サイトは専門的な保安処理がされてないVercelシステムを使ったページです。 個人的に使用してるID又はPASSWORDの使用はなるべく控えてください。 ※サインアップの際書いてもらったIIDX ID及びe-mailは本人確認用のためであり、その他利用しません。 ※IIDX IDに1個のIDだけ加入が可能であり、重複加入は許容されていません。 サインアップを提出したら3~7日以内で管理者が確認の次第承認を行います。その後からVoteの機能が使用可能です。

What is DPOPz (Double Play Optionz)?

DPOPz is a Chart option statistics site for Double Players. It is designed to serve as an option placement guideline for other DPers, especially beginners, through community voting and submissions.

How to use

  1. Top
    Search for a song title using the search feature, and the option distribution for that song will be displayed immediately. You can also submit your own vote via the Vote button. (Sign-up is required to Vote.)
  2. Table
    All collected data can be browsed here. The data currently being tracked is as follows.
    LevelTarget ChartsStatus
    ☆12All AC ChartsActive
    ☆11All AC ChartsActive
    ☆10AC ANOTHER / LEGGENDARIA ChartsPending*
    ☆9AC ANOTHER / LEGGENDARIA ChartsPending*
    *To be added in the future
  3. Report
    If you encounter any issues on this site, please submit a report via this page.

Vote機能について (サインアップ必)

Clicking the Log In button or the Vote button activates the login function and requires a sign-up procedure to use the Vote function. After submitting an optional opinion in Vote, it will be reflected in DB for approximately 3-7 days and applied to statistics. ※This site uses a Vercel system that does not have specialized security processing. Please refrain from using your personal ID or PASSWORD as much as possible. ※The IIDX ID and e-mail you wrote at the time of signing up are for identification purposes, and I will not use it for anything else. ※Only one ID can be subscribed to the IIDX ID, and duplicate subscriptions are not allowed. ※Once you submit your sign-up, the administrator will approve it as soon as it is confirmed. The Vote feature will then be available.
